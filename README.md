# Generator for Spring Projects

[Yeoman](https://yeoman.io) generator for Spring projects.

## Usage

```sh
git clone https://github.com/akullpp/generator-spring.git
cd generator-spring
npm link
npx yo spring
```

## Stack

### Production

- PostgreSQL
- SpringDoc OpenAPI
- Spring Data JPA
- Spring Starter Web

### Development

- Flyway
- JUnit 5
- Lombok
- MapStruct
