# Generator for Spring Projects

## Usage

```sh
git clone https://github.com/akullpp/generator-spring.git
cd generator-spring
npm link
yo spring
```

## Stack

### Production

- Spring Data JPA
- Spring Starter Web
- PostgreSQL
- Swagger with UI

### Development

- Spring Configuration Processor
- Flyway
- Lombok
- MapStruct
- JUnit 5
- AssertJ
- Mockito
